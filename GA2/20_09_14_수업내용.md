- #include "stdio.h" - 다른  헤더파일을 읽어오겠다 라는 참조?

- %f : 실수(float)
- character : 문자  /  string :  문장  /  '' :  문자 적을 때 / "" : 문장 적을 때(맞나?)

- putchar : 문자출력  /  getchar : 문자읽기

- int g_     : 전역변수 선언.
-scanf 주의점 :  space를 읽기 중지시킴. / gets : 공백 포함 모두 읽기 가능. 띄어쓰기 해도 읽기 가능.
